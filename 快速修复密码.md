# 快速修复密码问题

## 问题
使用 admin / 123456 登录时报错"用户名或密码错误"

## 快速修复（3步）

### 步骤1：执行 SQL 脚本
```bash
mysql -u root -p agricultural_db < database/reset_passwords.sql
```

### 步骤2：确认执行成功
执行后应该看到类似输出：
```
Query OK, 3 rows affected
```

### 步骤3：重新登录
1. 刷新前端页面
2. 使用以下账号登录：
   - 用户名：`admin`
   - 密码：`123456`

## 如果还是不行

### 检查数据库中的密码哈希
```sql
USE agricultural_db;
SELECT username, password FROM users WHERE username = 'admin';
```

## 验证密码哈希是否正确

如果你有 Java 环境，可以运行：
```bash
cd backend
mvn exec:java -Dexec.mainClass="com.agricultural.util.PasswordGenerator"
```

这会生成一个新的密码哈希值，你可以用它来验证。
